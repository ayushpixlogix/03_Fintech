# Fintech Landing Page

A responsive fintech landing page built with HTML, CSS, and JavaScript.

## Overview

This project is a front-end website template for a fintech product. It includes:

- A hero section with product messaging and CTA
- Feature highlight blocks
- Responsive navigation for mobile devices
- Smooth UI behavior with jQuery

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- jQuery

## Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Run the site by opening `index.html` in your browser.

Optional (recommended for local development):

- Use a local server extension such as Live Server in VS Code for better asset loading and live reload.

## Main Files

- `index.html`: Main landing page
- `default.html`: Alternate/static page entry
- `css/style.css`: Core styling
- `css/responsive.css`: Responsive breakpoints and mobile styles
- `js/general.js`: UI interactions (menu toggle, sticky header, tabs, accordion, utility handlers)

## Notes

- Google Fonts are loaded from CDN in the HTML.
- Images and icons are stored in the `images/` directory.
